# 培養利益計算

ドラゴンクエストXの輝晶培養・閃輝培養向け利益計算ツールです。

## 主な機能

- 輝晶核利益計算
- 閃輝晶核利益計算
- バザー販売対応
- モーモンバザー販売対応
- ドロップ履歴記録
- 利益推移グラフ
- ドロップ割合グラフ
- 7餅 / 5餅期待利益計算
- 詳細計算表示
- スマホPWA対応

## 使い方

### 価格設定

- 核価格
- 破片価格
- 細胞価格

を入力してください。

### ドロップ入力

| ボタン | 内容 |
|----------|----------|
| 45 | 破片45個 |
| 75 | 破片75個 |
| 核 | 核直ドロップ |

PT全員分のドロップを入力してください。

### 販売方法

- バザー販売（手数料5%）
- モーモン販売（手数料なし）

を切り替え可能です。

### 期待利益

輝晶核モード

- 7餅（28戦）

閃輝晶核モード

- 5餅（20戦）

を基準として期待利益を表示します。

### 詳細計算

「詳細計算を表示」から

- 期待破片数
- 期待核数
- 期待売上
- 細胞代
- 期待利益

の計算過程を確認できます。

## 利用規約
